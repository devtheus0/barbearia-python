# 💈 Sistema de Barbearia

Sistema feito em Python para gerenciamento de uma barbearia.

## 🚀 Funcionalidades
- Cadastro de clientes
- Cadastro de serviços com preço
- Agendamento por dia e horário
- Cancelamento de agendamento
- Faturamento diário e mensal
- Ranking de serviços
- Agenda do dia e do mês

## 🛠 Tecnologias
- Python
- JSON (armazenamento de dados)

## 📌 Como executar

```bash
python sistema_barbearia.py
